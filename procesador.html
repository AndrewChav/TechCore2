PROCESADOR (CPU)

EL CEREBRO DEL SISTEMA

El CPU (Unidad Central de Procesamiento) es el encargado de realizar todos los cálculos matemáticos y lógicos. Imagina que es el "Jefe de Obra": da las órdenes a los demás componentes.

Este modelo utiliza una arquitectura Híbrida: tiene "Núcleos de Rendimiento" (P-Cores) para tareas pesadas como juegos, y "Núcleos de Eficiencia" (E-Cores) para tareas de fondo como el correo o antivirus, ahorrando energía.

DATOS TÉCNICOS

  • Velocidad de Reloj: 6.2 GHz (Ciclos por segundo: 6.2 mil millones).
  • Núcleos: 24 Núcleos Físicos (8 Potentes + 16 Eficientes).
  • Litografía: 2 nanómetros (Transistores microscópicos).
  • Caché L3: 128MB (Memoria ultra-rápida interna).
  • TDP: 253W (Calor generado bajo carga máxima).
